4. Репозиторий maven https://mvnrepository.com/, выбрать зависимость spring-core, скопировать последнюю версию зависимости, тоже сделать для spring-beans
тоже сделать для spring-context, вставить их в файл pom.xml, выбрать Enable-auto-import. В директории main создать директорию resources и пометить эту директорию, навести курсор и нажать правую кнопку мыши, выбрать mark directory as, выбрать recources root. Создать в директории main папку java, также пометить cources root.
